Pemrograman Berbasis Web

Pemrograman berbasis web adalah proses pembuatan pengembangan aplikasi yang berjalan diatas web browser atau server menggunakan bahasa pemrograman. Ini melibatkan penggunann teknologi seperti HTML, CSS dan Javascript untuk membuat halaman web, serta bahasa sisi server seperti PHP atau Python untuk mengelola interaksi pengguna dan data.

Capaian Pembelajaran

Mahasiswa diharapkan memiliki kemampuan sebagai Front-End Web Developer yang mampu membangun website statis dan dinamis, menerapkan prinsip-prinsip desain responsif, menggunakan tools modern seperti Git, Bootstrap dan React dan berkolaborasi dalam tim menggunakan Github serta mendeploy website ke internet.
